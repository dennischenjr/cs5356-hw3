An html element we have not talked about is the details element → Allows us to toggle visibility and create a widget that when I click, expands to reveal hidden content. I used it with summary, so I can expand on more details of my projects below.

A CSS property we have not talked about is the transition property → Allows us to create smooth animations between CSS property changes instead of applying them instantly. I used it with transform and color, so my profile image smoothly enlarges when hovered over, and my contact links change color gradually for a more polished effect.
